# Política de Custodia de Evidencia 

## 1. Objetivo 
Garantizar integridad, trazabilidad y validez probatoria. 

## 2. Reglas 
- Ningún archivo se borra 
- Cada archivo tiene hash SHA256 
- Cada acción queda registrada 
- Acceso restringido por rol 

## 3. Cadena de custodia 
Debe registrar: 
- Carga 
- Consulta 
- Descarga 
- Asociación a entregables 

## 4. Seguridad 
- Control de acceso 
- Auditoría 
- Hash maestro al cierre
