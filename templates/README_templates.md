# Templates de entregables MACAPA

Cada carpeta contiene:

- `<CODIGO>.pdf.txt` → Texto base para PDF/Word.
- `<CODIGO>.json` → Estructura para backend.
- `<CODIGO>.md` → Versión Markdown para UI / IA.

Los contenidos detallados de cada template ya están definidos en la conversación previa (AIP, EII, PAI, EID, OMB, REM, FFI, SAB, SIR, LPD, FFR, AVF, SBR, OMX, SRM, APX, CCE, CRM, CCR, LOD, CRP, RMR, CGA, RRR, CVR, FHR, LER, OAR, DDFR, OKR, PER, BNA, OIP, LEE, LRA, LSB).

El desarrollador puede:
- Copiar cada bloque `*.pdf.txt` como base de plantillas PDF.
- Usar cada `*.json` como esquema de modelo.
- Usar cada `*.md` como plantilla de UI o texto generado por agentes.
